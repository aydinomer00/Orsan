# Orsan Reminder

Basit ve komik tek sayfa React uygulaması: Kaptan Örsan’a her gün 13:00’ten önce raporu hatırlatır.

## Kurulum

```
npm create vite@latest orsan-reminder -- --template react
cd orsan-reminder
npm install
npm run dev
```

## Özellikler

- Deniz/kaptan teması, emoji ve hafif animasyonlar
- 13:00’a geri sayım ve durum mesajları
- “Raporu Gönderdim Kaptan!” butonu + emoji konfetisi
- Bahane üretici ve mürettebat checklisti
